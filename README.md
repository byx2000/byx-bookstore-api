# ByxBookstore

ByxBookstore是一个电子书在线浏览网站，包括用户登陆、用户注册、电子书查询、
收藏、评论、点赞等功能。

技术选型：

* 前端：Vue、Element-UI
* 后端：SpringBoot、MyBatis、SQLite

数据来源：掌阅iRead（通过Python爬虫爬取）
